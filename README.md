# retail-sales-analytics

## Overview

This project analyzes retail sales transactions using MySQL, SQL analytics, Python, pandas, and data visualization techniques to identify customer behavior, product performance, regional sales trends, and business KPIs.

---

## Technologies Used

- MySQL
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

---

## Features

- SQL-based analytics
- Revenue KPI analysis
- Customer ranking using window functions
- Regional sales analysis
- Monthly sales trend analysis
- Product performance analysis
- Correlation analysis
- Data visualization

---

## SQL Concepts Used

- GROUP BY
- Aggregations
- Window Functions
- Ranking Functions
- Analytical Queries

---

CSV Dataset

MySQL Database
↓
SQL Analytics
↓
Python + pandas
↓
Visualizations

Business Insights

---

## Business Insights

- Certain regions contribute significantly higher revenue.
- Discounts negatively affect profitability.
- A small percentage of customers generate major sales.
- Technology products contribute strong revenue performance.
